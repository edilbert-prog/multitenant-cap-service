# Multitenant CAP Service

This project is a multi-tenant application built using SAP CAP (Cloud Application Programming Model).

## Project Structure

srv/
service.cds - CDS service definition
service.js - Service implementation

## Installation

Install dependencies:

npm install

## Run the service

cds run

The application will start locally.

## Technologies

- Node.js
- SAP CAP
- CDS
